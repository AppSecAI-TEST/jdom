JDOM2

Java manipulation of XML made easy.

This particular web site is intended for artifacts useful to the JDOM2 project. Most important things should b easy to find from this page.

The first thing you should do is read about The Updated Features in JDOM2. This will give you some idea of what is new, and what to expect. The document should be changing relatively frequently as features are added, so check out the change history log for it too. Currently the following are available:
  • Downloads of the JDOM2 snapshots. These snapshots are released intermittently as an when new features become available. The snapshot description should have more details for each release.
  • The JavaDoc of the most recent snapshot.
  • jUnit Test Report of the most recent snapshot.
  • Cobertura Test Coverage Report of the most recent snapshot.
  • Performance Metrics giving some idea of the relative performance of JDOM (relative to itself over time).
  • JDK Performance gives some idea of the relative performance of different JDK versions.
