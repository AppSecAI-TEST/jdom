JDOM2

Java manipulation of XML made easy.

This particular web site is intended for artifacts useful to the JDOM2 project.

Currently the following are available:
  • Downloads of the JDOM2 snapshots
  • The JavaDoc
  • jUnit Test Report
  • Cobertura Test Coverage Report
  • Performance Metrics

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/hunterhacker/jdom

get the source code on GitHub : hunterhacker/jdom
